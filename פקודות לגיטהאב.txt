פקודות לGIT

git add . - כל הקבצים שנעשה בהם שינויים, השינוי ישמר בקובץ נסתר שאני לא רואה. 
git commit -m "write a coment"- מצמיד את ההערה לשינוי
git push origin master הענף הראשי נקרא מאסטר". זו הפקדוה שמעלה אתה שינוי לגיטהאב

git remote -v - זה מראה אם חיברתי את התיקייה לגיטהאב
 git checkout -b new_branch - פותח ענף חדש (השם זה במקום new_branch)

git checkout (branch name) - לחזור לענף כלשהו 
git checkout - b (brnach name) - זה ליצור ענף חדש

git merge (branch name) - לאחד את הענף הקיים עם ענף קודם לו(שם הענף זה הענף אליו נאחד)
